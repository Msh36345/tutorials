## 🗂️ מבנה התוכן

כל הקבצים מכסים את אותם נושאים:

### 📅 **חלק 1: עבודה עם תאריכים ושעות**
- ניתוח תאריכים אוטומטי
- המרת טקסט לתאריכים
- פירוק תאריכים לרכיבים (שנה, חודש, יום, שעה)
- סינון לפי תאריכים וטווחים
- עבודה עם אזורי זמן
- חשבון עם תקופות זמן (Duration)

### 📈 **חלק 2: חלונות מתגלגלים (Rolling Windows)**
- ממוצע נע פשוט
- חישוב ממוצעים יומיים
- סטטיסטיקות מתגלגלות (min, max, std)
- שיטות שונות לחישוב חלונות
- Rolling context
- ויזואליזציה של ממוצעים נעים
- פונקציות מותאמות אישית

### ⏱️ **חלק 3: דגימה מחדש (Resampling)**
- תת-דגימה (Downsampling)
- דגימת יתר (Upsampling)
- אינטרפולציה למילוי פערים
- מילוי פערים עם datetime_range
- טכניקות מתקדמות

### 🔮 **חלק 4: חיזוי סדרות זמן (Forecasting)**
- מבוא לחיזוי סדרות זמן
- הכנת נתונים לחיזוי
- פיצול כרונולוגי train/test
- אימון מודלים (Linear, LightGBM, XGBoost)
- הערכת דיוק עם MASE
- Feature Engineering מתקדם
- ויזואליזציה של תוצאות

## ⚠️ שגיאות נפוצות ופתרונות

### שגיאה: `FileNotFoundError`
**בעיה:** קובץ הנתונים לא נמצא
**פתרון:** עדכן את הנתיב בקוד או העתק את הקבצים למיקום הנכון:
- `toronto_weather.csv`
- `historical_temperatures.csv` (לחיזוי)

### שגיאה: `ModuleNotFoundError: No module named 'polars'`
**בעיה:** Polars לא מותקן
**פתרון:** 
```bash
pip install polars
```

### שגיאה: `ModuleNotFoundError: No module named 'functime'`
**בעיה:** Functime לא מותקן (נדרש רק לחיזוי)
**פתרון:** 
```bash
pip install functime
```

### שגיאה: `Column 'datetime' is not sorted`
**בעיה:** שכחת להוסיף `set_sorted()`
**פתרון:** הוסף `.set_sorted('datetime')` לפני פעולות זמן

### הקוד רץ איטי
**בעיה:** שימוש ב-`rolling_map` או `collect()` מוקדם מדי
**פתרון:** השתמש בפונקציות מובנות ו-LazyFrame כמה שיותר
