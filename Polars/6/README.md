## 📝 דוגמה מהירה

```python
import polars as pl

# טעינת נתונים
df = pl.read_csv('data.csv')

# סינון ביקורות שמתחילות ב-"Very"
result = (
    df
    .filter(pl.col('content').str.starts_with('Very'))
    .select('content')
    .head()
)

print(result)
```
